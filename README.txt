Description: WooCommerce User Analysis tracks the subscriber's engagement for individual products on your woo-commerce site.


Why Use Woocommerce User Analysis plugin ★★★★★

Woocommerce User Analysis plugin helps you to track the user activity on the woo commerce Product page, It enables you to view the subscribers' engagement on the product.

It comes with a feature of sending an instant email to the user for remarketing the interested product or offer suitable discounts maunaly.


== Feature ==
- Compose email using shortcodes.
- Auto fetches the product details using shortcode.
- Send customized mail to the subscribers.
- perfect remarketing tool for your woo-commerce site.
